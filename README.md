# Bilibili-Vup-Stream-Details
a firefox browser extension for showing more vup stream details on bilibili

## !! VUP ONLY !!

vup (virtual up) 虛擬主播

由於只是從 vup.darkflame.ga 爬取數據，因此僅對虛擬主播有效。
